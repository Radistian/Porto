Bahasa Pemograman Yang Digunakan

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membangun dan menyusun konten pada halaman web. HTML memberikan struktur dasar untuk halaman web dan memungkinkan penambahan elemen-elemen seperti teks, gambar, tautan, formulir, dan banyak lagi.

CSS (Cascading Style Sheets) adalah singkatan dari bahasa Inggris yang berarti "Cascading Style Sheets". CSS merupakan bahasa desain yang digunakan untuk mendefinisikan tata letak (layout), warna, dan gaya tampilan elemen-elemen pada halaman web yang ditulis dalam bahasa markup seperti HTML.

JavaScript adalah bahasa pemrograman tingkat tinggi yang dikenal sebagai bahasa scripting. Bahasa ini biasanya digunakan untuk membuat dan mengontrol konten dinamis di halaman web. JavaScript bersifat client-side, yang berarti ia dijalankan pada sisi klien atau peramban web pengguna. Dengan JavaScript, Anda dapat membuat interaktivitas pada halaman web, mengubah konten secara dinamis, dan merespons tindakan pengguna.
